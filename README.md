**Klayout** 是一个免费开源的软件工具包，其底层调用pya库来设计GDAII版图文件。

## 关键特性

-   **免费开源** ；
-   采用纯Python语言编写，利用面向对象的特性来设计图形类；
-   目前可添加矩形、任意多边形、弧形、扇形四种基本图形；
-   每种图形可进行平移、旋转、缩放和镜像变换；
-   图形可嵌套，同一父类下的图形可执行相同的变换；

## 案例说明
-  五尖端端面耦合器：绘制了五条长度不同的端面耦合器，其边缘有切割标记；
-  双层SiN光栅耦合器：采用扇形和弧形来绘制；
